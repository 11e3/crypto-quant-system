# Development Tools

This directory contains development and analysis tools.

## Files

- `compare_trades.py`: Compare trade history between legacy and new backtest engines
- `compare_metrics.py`: Compare performance metrics between different backtests
- `check_logic_errors.py`: Check for logic errors in backtest implementation
- `verify_strategy_logic.py`: Verify strategy logic consistency
- `verify_strategy_logic_detailed.py`: Detailed strategy logic verification

These are development tools and not part of the main CLI.
