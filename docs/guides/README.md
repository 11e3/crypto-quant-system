# 사용 가이드

이 디렉토리에는 시스템 사용을 위한 가이드 문서들이 포함되어 있습니다. 모든 가이드는 Python 3.14 및 uv 기반 워크플로를 기준으로 합니다.

## 📚 가이드 목록

### 필수 가이드
- **[시작 가이드](getting_started.md)** - 설치, 설정, 기본 사용법
- **[설정 가이드](configuration.md)** - 설정 관리 방법 및 우선순위
- **[전략 가이드](strategies.md)** - 제공되는 전략 상세 설명
- **[전략 커스터마이징](strategy_customization.md)** - 커스텀 전략 작성 방법

## 🚀 빠른 시작

1. **처음 사용하시나요?** → [시작 가이드](getting_started.md)부터 시작하세요
2. **설정이 필요하신가요?** → [설정 가이드](configuration.md)를 참고하세요
3. **어떤 전략을 사용할지 모르시나요?** → [전략 가이드](strategies.md)를 확인하세요
4. **전략을 커스터마이징하고 싶으신가요?** → [전략 커스터마이징](strategy_customization.md)을 확인하세요

## 📝 추가 가이드 (예정)

- 데이터 관리 가이드
- 백테스팅 고급 기법
- 실시간 거래 봇 설정
- 모니터링 및 알림 설정
- 문서 빌드/배포 자동화 (Sphinx, GitHub Actions)
